⚽ ScoreFootball API

API para análise e previsões do Brasileirão baseada em dados reais. Gera probabilidades utilizando modelos estatísticos como Poisson e Monte Carlo.

⚡ Exemplo

{
  "time": "Palmeiras",
  "chance_titulo_%": 72.5,
  "chance_g4_%": 95.2,
  "chance_rebaixamento_%": 0.1
}

🚀 Começar

📄 Ver endpoints detalhados 🔍 Swagger UI

📊 Principais recursos

  • 📈 Projeção por média e frequência
  • ⚡ Power Ranking (ataque vs defesa)
  • 🎲 Simulação de Monte Carlo
  • 📅 Próximos jogos organizados por rodada

🧠 Modelos

Poisson: estima número de gols por time.

Monte Carlo: simula milhares de cenários do campeonato.

⚠️ Limitação

Os cálculos utilizam apenas dados da temporada atual.
